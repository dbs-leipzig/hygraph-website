Hygraph - Privacy Policy

The GDPR stipulates that you receive transparency about how and for what purpose your personal data is processed. That is why we are happy to explain to you here like those dates incurred when using this blog, are processed, and what they are used for.

Information We Collect

We do not collect any data. However the host may need to collecct data for some functional needs, please refer to the Hosting part for more informations

Hosting

The host operates a web server to deliver the pages of this blog.

Hosted by GitHub Pages
This blog is open source and is hosted at "GithHub Pages".
Github declares its handling of data in this privacy policy . "... we only collect the minimum amount of personal information that is necessary to fulfill the purpose of your interaction with us ..."
As a US company, GitHub meets the requirements of the Privacy Shield for users from the EU.

Hosted as a static website
This is a static website, ie the pages of this blog are ready-made on the host's web server and are delivered unchanged as soon as you click on a link to our blog in the browser (or enter a URL). With such an http request, your browser transmits your IP address and the path of the requested page. This is necessary so that the browser can load the requested page and deliver it to your browser.

Server logs
In addition to your IP address and the requested page, the http request also transmits the referrer (the page on which you clicked the link), information about your browser and operating system to the web server The host saves this information in server logs in order to monitor operation so that faults and cases of abuse can be quickly identified and rectified.
